# Práctica JavaScript Accesible

Este proxecto consiste nunha páxina web interactiva deseñada con criterios de accesibilidade, que integra diversas funcionalidades dinámicas mediante JavaScript.

**Autor:** Alejandro Mato

## 📄 Estrutura do Proxecto

- `index.html`: Estrutura semántica e accesible con formularios e seccións interactivas.
- `css/styles.css`: Estilos modernos, responsivos e con variables CSS.
- `js/script.js`: Lóxica para validacións, contadores e manipulación do DOM.

## 🚀 Funcionalidades Principais

- **Formulario de Contacto:** Con validación en tempo real mediante expresións regulares e contador de caracteres para a mensaxe.
- **Contador de Clics:** Sistema sinxelo para rexistrar interaccións do usuario.
- **Cor Aleatoria:** Cambio dinámico da cor de fondo cunha paleta clara.
- **Escoitador de Teclado:** Identificación da última tecla pulsada e contador de caracteres.
- **Caixa Interactiva:** Cambio de cor mediante eventos `mouseover` e `mouseout`.
- **Footer Interactivo:** Actualización automática do ano actual.

## ♿ Accesibilidade

- Uso de etiquetas ARIA (`aria-live`, `aria-describedby`, `role`).
- Etiquetas de formulario claras e elementos ocultos especificamente para lectores de pantalla (`sr-only`).
- Xestión de foco en caso de erros de validación.
